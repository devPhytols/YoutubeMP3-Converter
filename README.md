
# pConverter - Convert Youtube Videos to MP3

[pConverter] Basic system using ytdl-core to convert youtube videos to mp3. 

## How to Start

```
1 - git clone https://github.com/devPhytols/YoutubeMP3-Converter.git
2 - npm install
3 - node .
```
